deklaracja zmiennej
    stała - const nazwa zmiennej = wartość
    zmienna - let/var nazwa zmiennej = wartość

typy zmiennych
    let a = 4 - zmienna numerowa
    let b = "jakiś tekst" - zmienna typu string
    let c = 'jakiś tekst' - zmienna typu string

    let d = true - zmienna typu boolean, czyli typu logicznego
    let e = null - wartość typu null

sprawdzanie typu zmiennej  
    console.log(typeof(a)) - w przeglądarce prawy klawisz, zbadaj i zakładka consola
